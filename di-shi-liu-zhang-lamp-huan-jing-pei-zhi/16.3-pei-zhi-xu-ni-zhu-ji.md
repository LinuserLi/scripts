# 16.3：配置虚拟主机

1、编辑apache 主配置文件 /usr/local/apache2/conf/httpd.conf，找到这一行：  \#Include conf/extra/httpd-vhosts.conf ，将前面的注释井号去掉，表启用虚拟主机配置：

```bash
Include conf/extra/httpd-vhosts.conf
```
